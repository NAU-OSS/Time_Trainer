Code of Conduct

Expected Behavior

  All community members are expected to:
  
  - Be respectful and considerate in communication
  - Provide constructive feedback
  - Accept criticism gracefully
  - Focus on improving the project
  - Help create a positive and collaborative environment

Unacceptable Behavior

  The following behaviors will not be tolerated:

  - Harassment or discrimination of any kind
  - Personal attacks or insulting language
  - Trolling or disruptive behavior
  - Publishin private information without permission

Enforcement

  If a contributor violates this Code of Conduct, maintainers may:

  - Issue a warning
  - Request removal of inappropriate content
  - Temporarily or permanently restrict participation

  All enforcement desicions are at the discretion of the project maintainer

Reporting Issues

  If you experience or witness unacceptable behavior, please open an issue labeled:

  "CODE OF CONDUCT"

  Reports will be handled respectfully and confidentially when possible

Acknowledgement

  By participating in this project, you agree to follow this Code of Conduct

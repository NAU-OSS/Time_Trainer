Contributing to Time_trainer

Thank you for helping improve Time_trainer!

1. Reporting Bugs

Use the Issue tracker and title it with :

"BUG:"

Example:
BUG: Focus session timer resets unexpectedly

Be descriptive and include:
- What happened
- Steps to reproduce
- Your environment
- Screenshots if helpful

2. Suggesting Ideas

Use the Issue tracker and title it with:

"IDEA:"

Example: 
IDEA: Add habit streak tracking/

Explain: 
- What the feature is
- Why it improves the project


3. Contributing Code

1. Make an Issue first (BUG or IDEA)
2. Create a branch from that issue
3. Do your work.
4. Use commit prefixes: 
  -FIX: for bugs
  -FEAT: for features
  -DOC: for documentation
5. Open a Pull Request. 
6. Address review comments.
7. Reviewer merges the PR and closes the issue

4. Code Style

- Keep code clean and readable
- Add short docstrings explaining what functions do

5. Community

Follow the guidelines in CODE_OF_CONDUCT.md

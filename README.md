Time_trainer project is a Python-based training application.

If you would like to learn open source develpoment or contribute to this project, you are more than welcome to be apart of the comminity!

Requirements
-Python 3.11 or higher

Setup clone this repo:
git clone [https://github](https://github.com/NAU-OSS/Time_Trainer.git)
cd Time_Trainer

To install dependencies
pip install -r requirements.txt

Run the application
python time_master.py
